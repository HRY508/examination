# 工程简介
#### 介绍
在线考试项目，分为用户端和管理端，用户可在线开始，自我测评
管理员可通过管理系统对题目、试卷进行编辑，支持单选、多选、判断，
可设定定时考试，自动完成试卷判改
#### 软件架构
采用 SpringBoot+MybatisPlus+MySQL+Thymeleaf+Shiro
